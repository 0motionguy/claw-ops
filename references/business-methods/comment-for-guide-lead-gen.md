# Comment-For-Guide Lead Generation

## Overview
Instagram/TikTok lead generation strategy using "Comment X and I'll send you the guide" CTAs. Proven to drive high engagement and build qualified lead lists.

## The Pattern
1. **Hook** with valuable/problematic content
2. **CTA** "Comment '[KEYWORD]' and I'll send you the guide"
3. **Deliver** via DM with additional value
4. **Nurture** with follow-up content

## Examples from Analysis
- @adamstewartmarketing: "Comment 'MoltBolt' and I'll send you the guide" (1.3K likes, 352 comments)
- @ssssangha: "Comment 'BOT' and I'll send you the full guide"

## Why It Works
- Comments boost algorithm visibility
- Creates FOMO/curiosity
- Qualifies interested prospects
- Builds DM list for direct communication
- Simple keyword = easy to remember

## Implementation Steps

### 1. Choose Your Keyword
- Short (3-8 characters)
- Memorable
- Relevant to topic
- Examples: MOLT, BOT, GUIDE, FREE, CODE

### 2. Create Lead Magnet
- PDF guide
- Checklist
- Template
- Script
- Configuration file

### 3. Post Structure
```
[HOOK - Problem or Promise]
[BODY - Value/Warning/Tips]
[CTA - Comment 'X' for guide]
```

- DM conversion rate
- Guide download rate
- Email signup rate (if included)
- Sales conversion from leads
