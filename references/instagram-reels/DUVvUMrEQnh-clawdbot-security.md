## Content Summary
**10 Security Tips for OpenClaw/Moltbot/Clawdbot:**

1. **Don't run on main machine** — use cheap VPS or Mac Mini
2. **Never run as root** — limited user keeps hackers contained
3. **Change default port** — 18789 is known by all hackers
4. **Install Tailscale** — makes you invisible to internet
5. **SSH keys + Fail2ban** — passwords get guessed, keys don't
6. **Firewall with UFW** — close unused ports
7. **Allowlist your users** — only you can message it
8. **Ask bot to audit security** — "Check my logs for failed logins"
9. **Set up real-time alerts** — Claude monitors 24/7
10. **DMs only** — group chat = everyone can control it

**BONUS:** Scan every skill before installing. Hackers hide malicious prompts.

**Note:** Clawdbot = Moltbot = Openclaw (same tool, different rebrand names)
