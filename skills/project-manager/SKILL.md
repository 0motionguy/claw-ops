# Project Manager

Organize work, track deliverables, and manage team workflows.

## Supported Tools

### Notion
- Task databases
- Project wikis
- Team collaboration
- Custom workflows

### Trello
- Kanban boards
- Card-based tasks
- Automations
- Power-ups

### Asana
- Project timelines
- Task dependencies
- Team assignments
- Portfolio view

### Linear
- Issue tracking
- Sprint planning
- Git integration
- Fast performance

### Jira
- Agile workflows
- Bug tracking
- Custom fields
- Reporting

## Workflows

### Project Setup
```
1. Create project space
2. Define milestones
3. Set up task templates
4. Assign team roles
5. Configure automations
```

### Daily Operations
```
1. Review open tasks
2. Update statuses
3. Check blockers
4. Communicate updates
5. Track time
```

### Client Projects
```
1. Set up client workspace
2. Create milestone schedule
3. Weekly status reports
4. Deliverable tracking
5. Final handoff
```

## Scripts

- `scripts/project-creator.py` - Initialize new projects
- `scripts/task-sync.py` - Sync across platforms
- `scripts/status-reporter.py` - Generate updates
- `scripts/deadline-tracker.py` - Monitor due dates

## Best Practices

- Daily standups (async)
- Clear task descriptions
- Assigned owners
- Due dates on everything
- Regular retrospectives
