# Listicle Creator

Generate viral "X tips" content that gets saved and shared.

## The Formula

1. **Number** in title (3, 5, 7, 10, 21)
2. **Promise** clear benefit
3. **Deliver** scannable list
4. **CTA** for deeper dive

## Number Psychology

- **3-5 tips:** Quick wins, high completion
- **7 tips:** "Magic number" for memory
- **10 tips:** Comprehensive but not overwhelming
- **21 tips:** Ultimate guide positioning
- **101 tips:** Encyclopedia/reference

## Title Formulas

### Security/Warning
- "X things to stop [bad outcome]"
- "X mistakes that [problem]"
- "X warning signs of [issue]"

### Growth/Success
- "X ways to [desired outcome]"
- "X secrets to [success metric]"
- "X hacks for [benefit]"

### Learning/Skills
- "X things I wish I knew about [topic]"
- "X steps to master [skill]"
- "X fundamentals of [subject]"

## Structure

### Hook (3-5 seconds)
```
"10 things to [outcome]"
[Visual: Countdown or checklist]
```

### Tips (15-20 seconds each)
```
Tip 1: [Actionable advice]
→ Why it matters
→ How to do it
```

### CTA (5 seconds)
```
"Save this for later"
"Which tip will you try first?"
"Comment [KEYWORD] for the full guide"
```

## Engagement Boosters

### Controversial Tip
Include 1 tip that challenges conventional wisdom.

Example: 
"10 marketing tips...
7. Stop posting daily (quality > quantity)"

### Missing Tip
End with "What's tip #11? Comment below"

### Save CTA
"Save this post—you'll want to reference it later"
(Listicles have highest save rates)

### Challenge
"Try tip #3 for 7 days and report back"

## Visual Formats

### Instagram Reel
- Text overlay with tip number
- Progress bar showing tip X/10
- Trending audio or voiceover

### Twitter/X Thread
```
10 lessons from [experience]:

1. [Tip]
[Explanation]

2. [Tip]
[Explanation]

[Summary tweet]
```

### LinkedIn Post
```
I made [mistake] for [time period].

Then I discovered these 10 principles:

[Carousel with 10 slides]

[CTA to download full guide]
```

1. **All tips must be actionable**
2. **Keep them truly distinct**
3. **Logical flow (beginner → advanced)**
4. **Clear CTA**
5. **Visuals match tips**
