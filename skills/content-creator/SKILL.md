# Content Creator

Generate high-quality content at scale for any platform.

## Content Types

### Blog Posts
- Long-form articles (1000-3000 words)
- Listicles and guides
- Thought leadership
- SEO-optimized content

### Social Media
- Instagram captions
- Twitter/X threads
- LinkedIn posts
- TikTok scripts

### Email Content
- Newsletters
- Drip sequences
- Promotional emails
- Cold outreach

### Marketing Copy
- Landing pages
- Ad copy
- Sales pages
- Product descriptions

## Workflows

### Content Calendar Generation
```
1. Define themes for the month
2. Generate 30 content ideas
3. Create content for each day
4. Schedule in advance
5. Monitor performance
```

### SEO Blog Post
```
1. Research keywords
2. Create outline
3. Write draft
4. Optimize for SEO
5. Add internal links
6. Create meta description
```

### Viral Social Campaign
```
1. Identify trending topic
2. Create hook/angle
3. Write 5-10 variations
4. A/B test headlines
5. Schedule optimal times
```

## Templates

### Blog Post Structure
```
[Hook - Problem or promise]
[Introduction - Why it matters]
[Body - 3-7 main points]
[Examples/Case studies]
[Conclusion - Summary + CTA]
```

### Social Media Formats
**Instagram:**
- Hook (first line)
- Body (value)
- CTA (engagement)
- Hashtags (20-30)

**Twitter/X:**
- Strong opener
- Thread structure
- Visuals/screenshots
- Engagement CTA

**LinkedIn:**
- Professional hook
- Story/lesson
- Actionable advice
- Discussion question
