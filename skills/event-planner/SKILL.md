# Event Planner

Create memorable events that achieve your goals.

## Event Types

### Virtual Events
- Webinars
- Virtual conferences
- Online workshops
- Live streams

### Physical Events
- Conferences
- Meetups
- Workshops
- Product launches

### Hybrid Events
- Simulcast sessions
- Virtual networking
- Remote participation
- On-site experience

## Planning Process

### 3 Months Before
- Define objectives
- Set budget
- Choose date/venue
- Book speakers

### 1 Month Before
- Open registration
- Promote event
- Finalize agenda
- Test technology

### 1 Week Before
- Confirm attendees
- Send reminders
- Prepare materials
- Tech rehearsal

### Day Of
- Registration/check-in
- Session management
- Networking facilitation
- Real-time support

### Post-Event
- Follow-up emails
- Share recordings
- Gather feedback
- Analyze results
