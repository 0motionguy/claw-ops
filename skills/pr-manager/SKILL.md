# PR Manager

Get media coverage and manage public perception.

## Services

### Media Relations
- Press release writing
- Media list building
- Pitch development
- Interview coordination

### Product Launches
- Launch strategy
- Embargo management
- Reviewer outreach
- Launch event PR

### Crisis Management
- Issue monitoring
- Response strategy
- Stakeholder communication
- Reputation repair

### Thought Leadership
- Byline articles
- Speaking opportunities
- Award submissions
- Expert positioning

- Cision (PR platform)
