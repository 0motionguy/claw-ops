Create and grow professional podcasts.

## Episode Workflow

```
Pre-Production (Week 1):
- Book guest
- Research topics
- Prepare questions
- Test equipment

Production (Day of):
- Record episode
- Backup files
- Initial notes

Post-Production (Week 2):
- Edit audio
- Write show notes
- Create clips
- Design thumbnail

Publishing (Week 3):
- Upload to host
- Distribute to platforms
- Social promotion
- Email list
```
